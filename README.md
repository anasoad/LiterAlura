# LiterAlura - Catálogo de Livros 📚
   
O **LiterAlura** é uma aplicação Java desenvolvida como desafio do programa Oracle Next Education (ONE) em parceria com a Alura. O objetivo é criar um catálogo de livros que consome dados de uma API externa (Gutendex), armazena-os em um banco de dados relacional e permite consultas personalizadas via console.

## ✨ Funcionalidades

O sistema oferece um menu interativo com as seguintes opções:

1.  **Buscar livro pelo título**: Pesquisa na API Gutendex, salva no banco de dados e exibe os detalhes.
2.  **Listar livros registrados**: Exibe todos os livros que já foram salvos no banco de dados local.
3.  **Listar autores registrados**: Mostra a lista de todos os autores salvos no sistema.
4.  **Listar autores vivos em determinado ano**: Filtra autores que estavam vivos no ano informado pelo usuário.
5.  **Listar livros em determinado idioma**: Filtra as obras por siglas (ex: `en`, `pt`, `es`, `fr`).

---

## 🛠️ Tecnologias Utilizadas

* **Java 17**
* **Spring Boot** (Spring Data JPA)
* **PostgreSQL** (ou H2 para testes)
* **API Gutendex** (Dados sobre livros do domínio público)
* **Jackson** (Para desserialização de JSON)
